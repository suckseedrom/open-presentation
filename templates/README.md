# Templates

This folder holds on-demand template assets.

## Contract

- `templates/index.json` is the compact metadata layer.
- `templates/*/preview.md` is the generated visual preview card used during shortlist and selection.
- `templates/*/design.md` is the full design recipe used only after a template is chosen.
- `templates/presentation-feature-core/examples/` contains reusable base-template briefs.

Do not bulk-read every template `design.md`. Load only the one the user picks.
