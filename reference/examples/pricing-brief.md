# Example brief: Pricing / Package Presentation

Use `presentation-feature-video-ads` to turn pricing or package page content into a content-sized presentation.

## Input

- source page: pricing or package page
- goal: cinematic video ads presentation
- style: same house feel as the bundled PresentationFeature style guide
- constraints: preserve the player shell, preserve the premium motion, and use the smallest slide count that fully expresses the pricing story

## What success looks like

- the deck feels like a high-trust commercial story, not a spreadsheet
- it explains the logic behind pricing, not just the numbers
- it shows tiers, comparisons, and CTAs with the same premium presentation language
- it stays compact and content-sized instead of forcing a fixed slide count
