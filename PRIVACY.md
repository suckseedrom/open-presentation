# Privacy Policy

`open-presentation` is a public plugin and skill package for generating cinematic HTML presentations and video ads from briefs, pages, screenshots, notes, and related product materials.

## What this package does

The package provides bundled markdown workflow instructions, shared references, templates, examples, and a local player/export library. It is designed to run inside compatible AI agent apps and local workspaces.

## Data handling

This repository does not operate a hosted backend, telemetry service, analytics endpoint, or remote data pipeline of its own.

When installed from GitHub or copied locally, the package itself:

- does not transmit user content to an Allkons-managed server
- does not bundle a tracking SDK
- does not require account creation with Allkons
- does not store generated project data outside the host environment by default

## What data may still be processed

Your chosen AI host, IDE, browser, or plugin runtime may still process:

- prompts
- screenshots
- local files
- generated outputs
- export artifacts

That processing is controlled by the host product you use, not by this repository alone.

## Local files and generated output

Generated presentations, supporting assets, and export outputs are typically written to the current workspace or local browser context. Review your host environment before using sensitive material.

## Third-party services

This package may be installed through third-party products such as Codex, Claude Code, Cursor, OpenCode, GitHub, or a browser runtime. Those services have their own privacy terms and data retention policies.

## Security posture

This repository is intended to stay zero-dependency for the generated presentation itself and should not require hidden background services to function. Public docs and tests are kept aligned with that expectation.
